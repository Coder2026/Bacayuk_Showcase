# ShareSphere

Backend REST API for a location-based item rental platform.

## Tech Stack
- Java 21
- Spring Boot 3
- Spring Security (JWT)
- PostgreSQL
- Docker
- Gradle

## Main Features
- Authentication & Authorization
- Item Management
- Borrowing Workflow
- Review System
- Notifications
- File Upload

## Getting Started

```bash
./gradlew bootRun
```

Configuration values that contain secrets have been replaced with environment variable placeholders.
